job Brief
We are looking for a Full Stack Developer to build and maintain our web applications. 
You will work on both the front-end (what users see) and the back-end (the server and database) to deliver 
complete, high-quality software features.

 Key Responsibilities
 Write Code: Build user-friendly websites and strong server-side applications.
 Create APIs: Connect the front-end and back-end smoothly using APIs.
 Manage Data: Design and maintain databases to store app information safely.
 Test Apps: Find and fix software bugs to keep systems running fast.

 Requirements
 Experience: Proven work history as a Full Stack Developer.
 Front-End: Good skills in HTML, CSS, and JavaScript frameworks (like React or Vue).
 Back-End: Experience with server languages like Node.js, Python, or Java.
 Databases: Knowledge of database systems like SQL or MongoDB.
 main and extra skils: Docker ,AWS , REST APIs
